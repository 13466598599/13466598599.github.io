Mysql取消外键约束

2022年04月25日

SET FOREIGN_KEY_CHECKS=0;在Mysql中取消外键约束

Mysql中如果表和表之间建立的外键约束，则无法删除表及修改表结构

解决方法是在Mysql中取消外键约束: SET FOREIGN_KEY_CHECKS=0;

然后将原来表的数据导出到sql语句，重新创建此表后，再把数据使用sql导入

然后再设置外键约束: SET FOREIGN_KEY_CHECKS=1;
